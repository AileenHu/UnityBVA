# BVA_variable_collection

## 概览

将其他资源放入以后可能会使用的文件中。
现在，支持Textures,Cubemaps, Materials, AudioClips, AnimationClips, Lightmaps。


|              | 类型         | 描述            | 是否必需             |
|----------------|------------|---------------|----------------------|
|**type**            | `enum`      | 变量类型           | No    |
|**collections**               | `List<id>`      | 指向引用字段索引的集合         | Yes   |
